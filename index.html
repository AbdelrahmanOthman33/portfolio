EchoHandRobotics

Deaf and mute individuals often face barriers when communicating with people unfamiliar with sign language, limiting expression and understanding. With an estimated 70 million deaf people worldwide, inclusive solutions are vital. Our project addresses this gap through a wearable glove that enables two-way communication. First, the glove detects and interprets sign-language hand movements, converting them into audible speech so hearing users can understand without knowing the signs. Second, it incorporates speech-to-text, capturing spoken language and displaying it on an LCD for the deaf user to read. Together, these features translate sign into speech and speech into text, enabling smoother, accessible interactions.

Bridging the Communication Gap NLP

I developed an AI- and API-based system to render complex Arabic poetry, and other Arabic media such as films, videos, and comics, into accessible English while preserving meaning and cultural nuance. The platform combines Arabic speech/text ingestion with context-aware translation and a post-editing layer that resolves idioms, metaphors, and allusions, producing readable English prose or subtitles alongside literal glosses. By prioritizing semantic fidelity over word-for-word equivalence, it bridges a persistent communication gap between Arabic and English audiences.

Recruiter Agent APIs

Recruiter Agent is a toolkit that automates recruiting workflows across Gmail, Google Calendar, Outlook, Greenhouse, GitHub, Slack, and Zoom. It provides lightweight API wrappers for use in scripts or OpenAI agents. After installing dependencies, configure credentials via config files or environment variables (e.g., OUTLOOK_TOKEN, SLACK_API_TOKEN, GITHUB_TOKEN). Features include sending emails, managing calendars, scheduling interviews, creating candidates, querying GitHub, messaging on Slack, hosting Zoom meetings, and researching salaries (Levels.fyi, Glassdoor, etc.). Develop locally with a FastAPI backend and React frontend; run tests with unittest. A production Docker image serves the frontend via Nginx and proxies API requests under /api for easy deployment.

Faculty Coding Lab Overview EdTech

Faculty Coding Lab matters because it protects academic integrity while making assessment fair, fast, and scalable. Locked-down exams, verified identity, and gaze-based flags deter misconduct, so honest students are rewarded. Hidden unit tests and a standardized Docker runner deliver objective, reproducible scoring and instant feedback aligned to learning outcomes. Detailed session logs create an auditable trail that strengthens trust for students, instructors, and accreditation reviews. A simple instructor UI lowers grading workload, enables timely interventions, and exports evidence for appeals. Optional manual interviews preserve human judgment and inclusivity. The platform elevates rigor, equity, and transparency in programming education at scale.
